# 叙事设计

> 叙事结构、故事线、NPC弧、事件分层、多周目内容。

---

## 叙事结构

### 开场

革命成功后，序章用**互动蒙太奇**（3-5张象征性卡牌）交代革命过程：

> 「老少校的梦」→「蹄子起义」→「琼斯被驱逐」→「七诫刻上墙」→「第一次丰收」

玩家通过打出这几张牌"经历"革命，简洁、有仪式感。然后正式进入游戏。

### 整体结构

时间轴固定（2-3个游戏年），故事线网状展开。

- 第一年：分歧的起点（蜜月期 → 路线分化）
- 第二年：平行的真相（不同路线看到不同的农场）
- 第三年：收束（所有路线指向 Boxer 的命运）

[→ 参见 world-building.md — 时间框架详细设计]

## NPC 故事弧

每个重要 NPC 都有独立的故事弧（3-4个阶段），由玩家行为触发推进。

| NPC | 原著命运 | 游戏中的叙事弧 |
|---|---|---|
| **Boxer** | 累垮后被卖到屠宰场 | 核心情感线——他的信念如何被一步步摧毁（或不被摧毁） |
| **拿破仑** | 成为独裁者 | 权力线——从不同角度观察他的崛起，看到他的弱点和恐惧 |
| **Squealer（尖嗓子）** | 拿破仑的宣传官 | 谎言线——看穿他的宣传，被说服，还是利用他的谎言？ |
| **Snowball（雪球）** | 被驱逐，成为"叛徒" | 真相线——他的真实面目是什么？是英雄还是也有问题？ |
| **Benjamin（本杰明）** | 冷眼旁观的驴 | 镜像线——他是"如果你选择沉默会变成什么样"的镜子 |
| **Mollie（莫莉）** | 逃跑去和人类在一起 | 选择线——她的"背叛"真的是背叛吗？还是另一种自由？ |
| **Clover（三叶草）** | 母马，隐约察觉不对但不敢说 | 共鸣线——她是另一个"你"，也在犹豫要不要说出来 |
| **Sheep（羊群）** | 盲目跟从 | 群体线——试图唤醒他们，还是利用他们？ |

### NPC 故事弧示例：Benjamin

| 阶段 | 触发条件 | 叙事内容 |
|---|---|---|
| 一：沉默 | 默认 | 他对你沉默不语，偶尔用意味深长的眼神看你 |
| 二：暗示 | 亲近度 ≥ 友善 | 他开始用隐晦的方式告诉你一些事——"你有没有注意到墙上的字变了？" |
| 三：真话 | 信任度 ≥ 坦诚 | 他直接和你说真话，成为你的情报来源 |
| 四：选择 | 游戏后期 | 他做出选择——继续沉默，还是和你一起行动（取决于你整个游戏的行为模式） |

[→ 参见 core-mechanics.md — 关系系统如何驱动NPC弧]

## 事件分层设计

### 大事件（5-8个）

对应原著关键节点，所有玩家都会经历，但经历方式和选择不同：

1. 七诫刻上墙
2. 苹果/牛奶被独占
3. 雪球被驱逐
4. 风车计划争议
5. 风车被摧毁
6. 大清洗
7. 七诫被改为"所有动物一律平等，但有些动物更平等"
8. Boxer 被送走

> 每个大事件的具体分支结构待单独设计。

### 中事件（20-30个）

和特定故事线绑定的关键情节，不同路线的玩家经历不同的中事件。

### 小事件（50-100+个）

日常遭遇、随机触发、关系驱动的小场景。填充时间轴，让世界感觉活着。

### 微事件（系统生成）

文本层面的细节变化：同一张卡牌的描述在不同回合微妙不同，NPC 的对话根据关系值变化。

### 事件链示例：「苹果事件」

```
事件1：你发现猪在偷藏苹果
  → 选择A：不管 → 事件2A
  → 选择B：告诉其他动物 → 事件2B
  → 选择C：自己去质问猪 → 事件2C

事件2A（不管）：
  → 苹果被吃光，其他动物开始挨饿
  → 事件3：一匹小马因为饥饿晕倒
  → 选择：照顾小马 / 无视 / 告诉Boxer

事件2B（告诉其他动物）：
  → 动物们不满，但没人敢公开说
  → Squealer 发表演讲解释"猪需要营养"
  → 选择：公开反驳 / 私下继续传播 / 沉默
  → 根据魅力检定：部分动物被说服 / 大部分不信但开始怀疑

事件2C（质问猪）：
  → 拿破仑盯着你："你觉得呢？"
  → 获取卡选项：有「情报：猪的交易」→ 你知道拿破仑在和人类做交易
  → 属性选项：意志 ≥ 14 → 能坚持立场不退缩
  → 都不够 → 被迫退让
```

一条事件链有 10+ 个分支节点，产出不同的获取卡，影响不同的关系值。

## 多周目解锁内容

多周目的动力不只是"不同的选择"，而是全新的内容：

| 周目 | 解锁内容 |
|---|---|
| 第一周目 | 只能走部分路线，看到部分真相 |
| 第二周目 | 解锁新故事路线，看到猪的内部运作 |
| 第三周目 | 解锁反抗路线，尝试组织推翻拿破仑 |
| 第四周目 | 解锁雪球视角的回忆碎片，发现雪球也不完全是好人 |
| 后续 | 每一周目都有：新的获取卡、新的NPC故事弧阶段、新的事件链 |

## 叙事体量估算

| 内容类型 | 数量 | 说明 |
|---|---|---|
| 大事件 | 5-8 | 所有玩家经历的关键节点 |
| 故事主线 | 5-6 | 由意识形态+行动路线交叉决定 |
| NPC故事弧 | 7-8条，每条3-4阶段 | 每个重要NPC的独立叙事 |
| 事件链 | 20-30条 | 每条10+分支节点 |
| 中事件 | 20-30 | 故事线绑定的关键情节 |
| 小事件 | 50-100+ | 日常填充 |
| 结局 | 7-8 | 条件组合触发 |
| 每周目新内容 | 新卡牌+新事件链+新NPC阶段 | 驱动复玩 |

预估游戏时长：单周目 8-12 小时，多周目总计 20-30 小时。
