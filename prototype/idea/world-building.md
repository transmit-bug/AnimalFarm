# 世界观设定

> 基于原型一「第七条诫命」的世界观与角色设定。

---

## 玩家身份

### 核心设定

**一匹年迈的劳力马，Boxer 的同批老伙计。**

- 革命前就和 Boxer 一起在琼斯手下劳作，一起参加革命，一起建设农场
- 你们是最亲密的战友，但走向了不同的方向——Boxer 选择无条件信任，你选择用自己的眼睛去看
- 赵刚式角色：看得清，说得透，但拦不住。Boxer 是你的李云龙
- 核心动机：不是为了自己活下去，而是想救 Boxer——让他看到真相。但 Boxer 不认为自己需要被救

### 叙事功能

- 你是"活历史"，你的记忆就是真相的载体
- 你和 Boxer 白天不在一起干活，收工后能聚在一起——制造"想劝他但没时间"的紧迫感
- 你的岗位让你能接触农场的各个角落和所有动物（负责运送物资、到不同区域帮忙）
- 马的寿命有限，游戏的时间压力有了生理基础——你在变老，你的时间不多了
- 你的岗位不显眼，猪不会特别关注你——给了你"暗中行动"的空间

### 其他身份选项（保留，暂不展开）

- **无名老马** — 活历史，记忆 vs 遗忘主题
- **曾经的革命参与者** — 理想的背叛，建设者看着农场被夺走
- **被流放后归来的动物** — 外部视角，陌生化体验

## 时间框架

游戏跨越约 2-3 个游戏年。时间轴固定，但故事线网状展开。

**结构原则：** 时间轴是骨架，每个时间段内部是网状的。不是"第一章→第二章→第三章"的线性推进，而是多个可能的故事线并行，玩家经历哪条取决于之前的选择。

### 第一年：分歧的起点

表面上所有玩家都经历"蜜月期"，但玩家的选择已经开始分化。

**行动路线分化（隐性，由关系值和行为模式决定）：**

- **体制内路线** — 你亲近猪，成为拿破仑的耳目，看到权力运作的内部
- **民间路线** — 你亲近底层动物，看到动物们的真实苦难，开始暗中串联
- **旁观者路线** — 你独来独往，收集信息但不介入，看到全貌但无力改变

**意识形态分化（与行动路线交叉）：**

- **理想主义者** — 仍然相信动物主义的原初理想，试图在体制内维护它
- **实用主义者** — 不关心理想，只关心怎么让日子好过一点，愿意和猪妥协
- **反抗者** — 已经认定这个体制没救了，开始暗中策划推翻它
- **投机者** — 看到了权力的运作方式，决定利用它为自己谋利

**交叉组合产生 5-6 条可玩主线：**

| 组合 | 叙事定位 |
|---|---|
| 体制内 + 理想主义 | 从内部改变体制（最困难的路线） |
| 体制内 + 投机者 | 成为拿破仑的帮凶（最黑暗的路线） |
| 民间 + 反抗者 | 组织地下抵抗（最危险的路线） |
| 旁观者 + 实用主义 | 独善其身（最孤独但最安全的路线） |
| 其他组合 | 也有对应的故事内容 |

**第一年末关键分支点：雪球被驱逐**

- 体制内路线 → 被迫投票支持拿破仑
- 民间路线 → 试图保护雪球
- 旁观者路线 → 什么都没做，但看到整个过程

### 第二年：平行的真相

不同路线的玩家看到完全不同的农场：

- **体制内路线：** 看到猪的腐败、秘密会议、和人类的交易。离权力很近，但离 Boxer 很远
- **民间路线：** 看到动物们的饥饿、恐惧、互相告密。离 Boxer 很近，但离真相的核心很远
- **旁观者路线：** 看到全貌，但关系值不足以影响任何人。最清醒，也最无力

**第二年末关键分支点：Boxer 受伤/累垮**

- 体制内玩家：有机会用地位救 Boxer，但代价是暴露自己
- 民间玩家：可能已组织足够的动物来保护 Boxer，但 Boxer 自己不愿意被保护
- 旁观者玩家：什么也做不了，只能看着

### 第三年：收束

所有路线在第三年开始收束——不是因为故事变线性了，而是所有路线都指向同一个终点：Boxer 的命运。

但每条路线到达终点的方式不同，取决于：
- 手里有什么获取卡
- 和 Boxer 的关系值是多少
- 属性是什么状态
- 和其他动物的关系如何

这些条件组合决定了终章能做什么、不能做什么，最终触发哪个结局。
