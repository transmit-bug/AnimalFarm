# 驱动系统与结局

> 四层驱动系统、结局设计、系统间矛盾关系。

---

## 四层驱动系统

驱动玩家继续玩下去的不是单一动力，而是多层动力互相咬合、甚至互相矛盾。

### 第一层：生存压力（回合级驱动）

**"我今天能不能活下去？"**

最底层的驱动力。每个回合面临资源压力——食物、体力。资源不足时触发恶化事件链：

- 食物不足 → 偷其他动物的食物 → 关系值下降 → 被发现 risk 上升
- 体力耗尽 → 无法完成劳动任务 → 被猪标记为"不积极" → 政治风险上升
- 士气崩溃 → 某些思维卡牌被锁定，无法打出

特点：紧迫感强，但天花板低。玩家很快掌握基本节奏后，注意力转移到更高层。

### 第二层：真相拼图（章节级驱动）

**"到底发生了什么？真相是什么？"**

核心叙事驱动力。农场的现实在被不断篡改——七诫在变，历史在被重写，动物们在"消失"。真相通过碎片浮现：

- 谷仓墙角被涂掉的旧诫文
- 老马深夜低声告诉你它记得的"真实版本"
- 偷听到猪的对话片段
- 某张事件卡的描述文字和上一周目见过的同一张卡**措辞不同**

每拼出一块真相，解锁新的卡牌、场景、行动选项。不是奖励"更强"，而是奖励"看得更深"。

特点：好奇心 + 愤怒。玩家逐渐意识到事情不对，产生强烈动机去 uncover 全部真相。

### 第三层：身份演化（全局驱动）

**"我正在变成谁？"**

最深层的驱动力。牌组就是灵魂——随着游戏推进，获得的新卡牌反映行为模式：

- 总是选择对抗？牌组出现更多「反抗」「煽动」「牺牲」类卡牌
- 总是选择顺从？牌组出现更多「服从」「沉默」「自欺」类卡牌
- 暗中收集信息？牌组出现更多「洞察」「推理」「秘密」类卡牌

到游戏中后期，牌组已经定义了"你是什么人"。游戏会通过关键场景把镜子举到你面前——你看到自己的牌组，突然意识到：我已经变成了这样的人。

**核心矛盾：生存和身份是矛盾的。**

- 要活下去，可能需要顺从猪的命令（获得配给、避免惩罚）
- 但每次顺从，牌组就往"服从"方向偏移
- 等终于有了反抗的能力（资源充足了），牌组里已经没有反抗卡了
- **活下来了，但已经不是当初想成为的那个动物了**

### 第四层：命运解锁（周目级驱动）

**"如果当初做了不同的选择？"**

通关一次后，带着对故事的理解进入下一周目：

- 新卡牌解锁 — 上一周目因为牌组偏移而没见过的卡牌
- 新场景链解锁 — 因为知道了某个真相，某些之前沉默的NPC现在会说出不同的话
- 新分支解锁 — 关键节点出现之前不存在的选项

特点：知识即能力。不是数值变强了，而是作为玩家理解更深了，游戏也因此展开更多内容。

## 四层驱动的咬合关系

```
┌─────────────────────────────────────────────┐
│  第四层：命运解锁（周目）                       │
│  "如果当初……" → 新周目                        │
│  ┌───────────────────────────────────────┐  │
│  │  第三层：身份演化（全局）                 │  │
│  │  "我在变成谁？" → 牌组即人格             │  │
│  │  ┌─────────────────────────────────┐  │  │
│  │  │  第二层：真相拼图（章节）          │  │  │
│  │  │  "真相是什么？" → 碎片解锁         │  │  │
│  │  │  ┌───────────────────────────┐  │  │  │
│  │  │  │  第一层：生存压力（回合）   │  │  │  │
│  │  │  │  "能活下去吗？" → 资源管理  │  │  │  │
│  │  │  └───────────────────────────┘  │  │  │
│  │  └─────────────────────────────────┘  │  │
│  └───────────────────────────────────────┘  │
└─────────────────────────────────────────────┘

矛盾点：
  生存 ↔ 身份：活下去可能意味着变成你讨厌的人
  真相 ↔ 生存：知道真相可能让你陷入危险
  身份 ↔ 真相：你的牌组可能阻止你看到某些真相
```

## 结局系统

游戏不一定要打到最后一回合才结束。达到特定条件组合时触发结局。

### 7-8 个主要结局方向

| 结局 | 触发条件概要 | 情感基调 |
|---|---|---|
| **悲剧结局** | 资源连续耗尽，体力崩溃 | 你倒下了，和 Boxer 一样 |
| **清洗结局** | 关键 NPC（拿破仑）关系值降至敌对 | 你被驱逐或消灭 |
| **沉默结局** | 意志属性持续下降至阈值以下 | 你放弃了思考，成为沉默的大多数 |
| **同化结局** | 顺从类获取卡占比 > 70%，魅力达到阈值 | 你变成了下一个压迫者 |
| **反抗结局** | 收集关键真相卡 + 意志 ≥ 14 + 特定关系条件 | 你站了出来（可能成功，可能失败） |
| **逃亡结局** | 特定关系卡 + 洞察达到阈值 + 找到逃跑路线 | 你离开了农场，但真相随身带走 |
| **真相结局** | 收集全部真相碎片 + 特定 NPC 信任达到最高 | 你揭露了全部真相（但可能无人相信） |
| **牺牲结局** | 特定条件组合，主动选择牺牲自己保护他人 | 最悲壮但最有尊严的结局 |

### 结局设计原则

- 每个结局由**属性要求 + 特定事件完成 + 特定卡牌持有 + 最终决策**组合触发
- 结局不是"好/坏"的二元判断，每个都带有苦涩和复杂性
- 不同结局分布在不同故事路线上——不是所有结局对所有路线开放
- 结局条件本身也是策略目标——玩家会为了达成某个结局而主动调整行为模式

### 结局与故事线的关系

| 结局 | 可达成路线 |
|---|---|
| 悲剧结局 | 所有路线 |
| 清洗结局 | 民间路线、反抗者路线 |
| 沉默结局 | 所有路线 |
| 同化结局 | 体制内路线、投机者路线 |
| 反抗结局 | 民间路线、旁观者路线 |
| 逃亡结局 | 旁观者路线、民间路线 |
| 真相结局 | 所有路线（但条件极难） |
| 牺牲结局 | 民间路线（需要高关系值） |

## 机制全景图

```
┌─────────────────────────────────────────────────────┐
│                    季节系统（规则层）                   │
│  春→夏→秋→冬，每季改变资源/体力/审查/可用行动          │
├─────────────────────────────────────────────────────┤
│                  回合循环（节奏层）                     │
│  晨起 → 规划(3AP) → 执行(行动卡→事件场景) → 日落 → 夜 │
├─────────────────────────────────────────────────────┤
│                  行动系统（操作层）                     │
│  每日行动卡(固定池) × 行动点(3+属性修正)               │
│  劳动 / 社交 / 调查 / 进食 / 休息                     │
├─────────────────────────────────────────────────────┤
│                  事件系统（叙事层）                     │
│  行动触发事件 → 三层选项(基础/获取卡/属性) → 结算       │
├─────────────────────────────────────────────────────┤
│                  策略系统（深度层）                     │
│  属性(D20检定) × 获取卡(手牌管理) × 关系值(双维度)     │
├─────────────────────────────────────────────────────┤
│                  压力系统（张力层）                     │
│  资源(文本状态) + 审查(定期+条件触发) + 卡牌藏匿        │
├─────────────────────────────────────────────────────┤
│                  驱动系统（动力层）                     │
│  生存 → 真相 → 身份 → 命运解锁                        │
├─────────────────────────────────────────────────────┤
│                  结局系统（目标层）                     │
│  7-8个结局，由属性+事件+卡牌+决策组合触发              │
└─────────────────────────────────────────────────────┘
```
