# 游戏整体定位

## 一句话描述

一款基于《动物庄园》的叙事驱动策略卡牌游戏，精神内核接近《极乐迪斯科》。

## 核心概念

- **类型：** 叙事驱动 + 策略卡牌
- **平台：** PC 独立游戏（Steam）
- **美术基调：** 暗黑童话 × 文学拼贴 × 冷峻寓言
- **叙事风格：** 碎片化叙事，但所有细节可推导还原
- **精神内核：** 权力腐败、理想变质、记忆与遗忘

## 核心体验

玩家扮演一匹年迈的劳力马，在革命成功后的农场中，眼睁睁看着理想被一步步篡改。
核心动机不是"活下去"，而是"让最信任的兄弟看到真相"——但他不认为自己需要被救。

## 设计原则

- 机制决定高度，叙事决定深度
- 玩法即信息（ludonarrative harmony）
- 做减法，不做加法——每个系统必须服务于核心体验
- 碎片不是随机的，是拼图

## 参考作品

| 作品 | 借鉴点 |
|---|---|
| 《极乐迪斯科》 | 思想即战场、技能检定、碎片化叙事、文学性 |
| 《Slay the Spire》 | 卡牌策略、手牌管理 |
| 《This War of Mine》 | 生存压力、道德困境 |
| 《Don't Starve》 | 暗黑童话美学 |
| 《Root》 | 阵营博弈、动物政治 |
| 《Hades》 | 多周目解锁、死亡即叙事 |
| 《Disco Elysium》 | 属性系统、内心声音、意识形态冲突 |
