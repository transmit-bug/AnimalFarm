# 核心机制设计

> 回合制、卡牌系统、属性系统、关系系统、资源系统、审查机制、季节系统。

---

## 一、回合流程

一个回合 = 一天。每回合玩家经历以下阶段：

```
1. 晨起（系统自动）
   → 睡眠恢复体力（少量）
   → 发放基础配给（食物状态更新）
   → 检查食物新鲜度（未食用可能腐烂）
   → 触发晨间事件（条件满足时）
   → 展示叙事文本 + 当前状态描述

2. 规划（玩家操作）
   → 查看今日行动卡（固定池）
   → 分配行动点（基础3点 + 属性修正）
   → 决定行动顺序

3. 执行（玩家操作，核心循环）
   → 逐张打出行动卡
   → 每张行动卡触发事件场景
   → 事件场景中使用获取卡决定深度
   → 结算选择结果

4. 日落（系统自动）
   → 结算今日所有效果
   → 关系值浮动
   → 条件事件检查（资源阈值/关系值/回合数）
   → 季节推进检查

5. 夜晚（可选，玩家选择是否进入）
   → 特殊事件（秘密集会、偷听、藏匿物品）
   → 消耗更多体力
   → 被发现 risk 更高
```

---

## 二、行动点系统

- 基础行动点 = 3
- 体力属性影响上限：体力低时 2 点，体力高时 4 点
- 「休息」不消耗行动点，但跳过当天剩余行动（直接到日落）
- 「进食」消耗 1 行动点

---

## 三、双层卡牌系统

### 每日行动卡（固定池）

- 每天固定存在，代表日常职责
- 消耗后次日刷新
- 基础行动卡：劳动 / 社交 / 调查 / 进食 / 休息
- 具体列表和效果待设计

### 获取卡（累积池）

- 通过事件获得，持久存在
- 每张卡有叙事来源（不是凭空出现的）
- 类型：
  - **情报卡** — 拼凑真相，解锁新场景/选项
  - **关系卡** — 影响特定动物的关系值，解锁互动选项
  - **思维卡** — 代表内心声音，打出后影响决策方式
  - **意识形态卡** — 反映政治立场，改变事件解读方式
- 每张卡有"危险等级"（1-3），影响审查时的没收概率

### 卡牌藏匿系统

- 危险卡牌可选择：随身携带 / 藏匿 / 传递给信任动物 / 主动销毁
- 藏匿需要消耗行动点
- 藏匿地点可能被其他动物发现
- 每个选择都是生存 vs 真相的矛盾

## 四、事件场景结构

行动卡打出后触发事件场景，场景结构：

```
① 叙事文本（系统根据条件生成/从预设中选取）

② 判定：是否需要属性检定？
   → 需要 → 掷 d20 + 属性修正值 vs 难度阈值
   → 不需要 → 直接进入选择

③ 选择阶段：三层选项叠加
   → 基础选项（所有人都能看到）
   → 获取卡选项（手中有相关卡才能看到/使用）
   → 属性选项（属性达到阈值才能解锁）

④ 结算：根据选择结果修改隐藏变量
   → 关系值变化
   → 资源变化
   → 获得/失去获取卡
   → 属性可能变化
   → 解锁/锁定后续事件
```

## 五、属性系统（DND 式）

角色拥有四项核心属性，通过行动和事件提升或下降：

| 属性 | 提升方式 | 影响日常 | 影响突发事件 |
|---|---|---|---|
| 体力 | 持续劳动、锻炼 | 劳动产出、行动点上限、夜间行动 | 逃跑、对抗、承受惩罚 |
| 洞察 | 观察、学习、偷听 | 观察深度、解读卡牌文字 | 识破谎言、发现隐藏线索 |
| 魅力 | 社交、演讲、帮助他人 | 社交效果加成、关系值变化幅度 | 说服、煽动、安抚 |
| 意志 | 抵抗、坚持信念 | 审查时卡牌保留能力、思维卡稳定性 | 面对威胁是否屈服 |

### 属性值与修正值

**属性值范围：** 3-18（DND 标准）
**属性修正值：** (属性值 - 10) / 2，向下取整

| 属性值 | 修正值 |
|---|---|
| 3 | -4 |
| 6-7 | -2 |
| 8-9 | -1 |
| 10-11 | 0 |
| 12-13 | +1 |
| 14-15 | +2 |
| 18 | +4 |

### 属性检定

**检定方式：** 掷 d20 + 属性修正值 vs 难度阈值

| 难度 | 阈值 | 叙事含义 |
|---|---|---|
| 简单 | 8 | 基本能成功 |
| 普通 | 12 | 需要一定能力 |
| 困难 | 16 | 需要突出能力 |
| 极难 | 20 | 几乎不可能，除非属性很高 |

### 属性规则

- 属性提升必须绑定叙事（不能自由加点）
- 属性可以下降（如每次屈服意志-1）

### 策略协同

属性 × 获取卡 × 关系值 三者协同构成完整策略空间：

- **属性** 决定"能做什么"
- **获取卡** 决定"做得多好"
- **关系值** 决定"做了之后别人怎么看你"

## 六、关系系统

每个重要 NPC 有两个维度：

| | 亲近 ← → 疏远 | 信任 ← → 警惕 |
|---|---|---|
| 含义 | 愿不愿意和你接触 | 愿不愿意和你说真话 |
| 影响 | 能否触发互动事件 | 互动时能获得多深的信息 |

### 数值范围

-3 到 +3，共 7 档：

**亲近度：** 敌对 · 冷淡 · 疏远 · 中立 · 友善 · 亲密 · 信赖
**信任度：** 警惕 · 怀疑 · 保留 · 中立 · 坦诚 · 信任 · 深信

### 关系值变化

- 社交行动 + 获取卡组合 → 改变关系值
- 事件中的选择 → 改变关系值
- 猪的宣传 → 可能降低其他动物对你的信任
- 时间流逝 → 不互动则缓慢向中立回归

### 关系值触发条件

- 亲近 ≥ 友善 且 信任 ≥ 坦诚 → 解锁该 NPC 的"秘密对话"事件
- 亲近 ≤ 冷淡 → 该 NPC 拒绝与你互动
- 信任 ≤ 怀疑 → 该 NPC 给你的信息是假的或残缺的
- 特定 NPC 关系值达到阈值 → 解锁关键剧情分支

[→ 参见 narrative-design.md — NPC故事弧如何由关系值驱动]

## 七、资源系统

- 基础配给保底 + 行动产出余量
- 资源状态用文本描述（饱足/温饱/饥饿/饥荒），不用数字
- 体力恢复与食物摄入挂钩
- 食物有新鲜度，不保存就腐烂
- 猪定期审查没收超额物资

## 八、审查机制

### 定期审查

每隔若干回合触发（具体频率由季节和政治局势决定）。

```
① 叙事：猪的巡逻队来了
② 判定审查强度（由政治局势决定）
③ 玩家选择应对：
   → 配合审查 — 安全，但被检查的卡可能被没收
   → 藏匿 — 提前藏好的卡安全，但藏匿行为有被发现 risk
   → 转移 — 把卡交给信任的动物，对方承受审查 risk
④ 检查携带卡牌：
   → 每张卡有危险等级（1-3）
   → 审查强度 vs 危险等级 → 决定是否被没收
   → 高洞察 → 能更好判断哪些卡危险
   → 高意志 → 审查时能保留更多卡
⑤ 结果结算
```

### 条件触发审查

- 关系值中"警惕"的动物数量多 → 有动物告密 → 额外审查
- 玩家持有高危险卡超过一定回合 → 猪得到情报 → 定向搜查

## 九、季节系统

| 季节 | 基础配给 | 体力恢复 | 特殊规则 |
|---|---|---|---|
| 春 | 正常 | 正常 | 新事件解锁（选举、规划），希望感最强 |
| 夏 | 正常 | +1 | 食物产出高，但猪控制加强，审查频率+1 |
| 秋 | 减少 | 正常 | 收获季，劳动产出翻倍，体力消耗也翻倍 |
| 冬 | 大幅减少 | -1 | 食物短缺，某些行动不可用，夜间体力消耗翻倍 |

季节推进由回合数自动推进（具体节奏待根据整体游戏时长确定）。
